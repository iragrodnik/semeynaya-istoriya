# План разработки сайта «Семейная история»

## Concept and Specification

### Цель
Создать современный, адаптивный (mobile-first) сайт-визитку для психологической группы, посвященной изучению семейной истории. Сайт должен вызывать доверие, быть информативным и обеспечивать легкую запись на курс.

### Технологический стек
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS (для быстрого и гибкого Mobile-first дизайна)
- **Deployment:** GitHub Pages (через GitHub Actions)
- **Icons:** Lucide React (для чистых и современных иконок)

### Структура контента (Single Page)
1. **Header:** Логотип/Название и навигация по якорям.
2. **Hero:** Эмоциональный заголовок, комбинированный анонс из обоих вариантов (максимальное вовлечение), кнопка CTA "Записаться".
3. **О группе:** Описание методики (ССТ), безопасного пространства и того, что получит участник.
4. **Темы встреч:** Список из 6 занятий (структурированный список или карточки).
5. **Тест:** Интерактивный блок «11 признаков здоровой семьи» (чек-лист с подсчетом результата).
6. **Ведущие:** Фото (заглушки), имена и специализация Анастасии Морозовой и Ирины Гродник.
7. **Орг. информация:** Дата старта, время, стоимость, ограничения по количеству мест.
8. **Footer:** Контакты (заглушки на Telegram/WhatsApp) и ссылки на соцсети.
